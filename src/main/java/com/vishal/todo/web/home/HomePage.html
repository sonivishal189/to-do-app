To-Do App is a application developed using Java, Apache Wicket, Hibernate, HTML and Bootstrap. This app provides feature to manage all tasks at one place. Various features provided are:

  • Create a new task for a building and assign it to a userDetail.
  • View all task list with task title, task owner and building name for which this task is created.
  • User can also view each task in details by selecting task and clicking on view task button which will show task id, task title, task assigned to, building for which task is created, who created task, what is the status of task, who updated it and also create and update data time.
  • User can also edit a task selecting task and clicking edit button.
  • User can delete 1 or tasks at a time by selecting task/s and clicking delete button.
  • This app internally updates the details of userDetail who created or updated a task along with timestamp.
